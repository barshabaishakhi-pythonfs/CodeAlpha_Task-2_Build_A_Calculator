# CodeAlpha_Task-2_Build_A_Calculator

A fully functional, responsive Calculator web application built using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development internship. This project features a clean, user-friendly interface with support for basic arithmetic operations.

## 🚀 Features
* **Core Arithmetic:** Handles addition, subtraction, multiplication, and division.
* **Responsive Design:** Optimized to work perfectly on desktops, tablets, and mobile devices.
* **Modern UI:** Built with clean layout styling, hover states, and button press feedback.
* **Clear & Delete Functions:** Easily clear the entire display or delete the last digit entered.

## 🛠️ Technologies Used
* **HTML5:** For structured semantic markup of the calculator layout.
* **CSS3:** For modern styling, button grids, and responsive alignments.
* **JavaScript (ES6):** For calculation logic, handling button inputs, and managing screen state.
